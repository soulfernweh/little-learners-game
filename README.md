# 🌟 Little Learners Adventure

A fun, educational game suite for 3-5 year olds that teaches basic skills through interactive play.

## 🎮 Games Included

### Main Game (`index.html`)
- **🔢 Counting** — Count objects (1-3 for age 3, up to 1-8 for age 5)
- **🎨 Colors** — Identify colors from a visual sample
- **⭐ Shapes** — Name shapes from emojis
- **🧠 Memory** — Classic card-matching pairs game
- **🦁 Animals** — Identify animals by sight or sound
- **🚗 Vehicles** — Name vehicles and where they go
- **🏠 Home Items** — Identify household objects and their uses

### Learn to Draw (`draw.html`)
Step-by-step guided drawing for kids — follow along to draw a Car, Truck, Bus, Ship, Airplane, or Helicopter.

### Magic Mirror (`mirror.html`)
Uses your camera + AI pose detection to animate a character that copies your movements in real-time! Choose from Stick Figure, Robot, Dinosaur, Fish, Cat, or Alien.

## ✨ Features
- Age-based difficulty (3, 4, or 5 years old)
- Voice encouragement using Speech Synthesis (says "Good job, [name]!")
- Gentle voice for wrong answers ("That's okay! Try again!")
- Streak-based excitement levels
- Sound effects and confetti celebrations
- Works on desktop, tablet, and mobile
- No installation required — just open in a browser

## 🚀 Play Online
Visit: https://soulfernweh.github.io/little-learners-game/

## 💻 Run Locally
Simply open `index.html` in any modern browser. No server needed.

## 📋 Requirements
- Modern browser (Chrome, Edge, Safari, Firefox)
- Camera access needed for Magic Mirror only
- Internet connection needed for Magic Mirror (loads AI model from CDN)
